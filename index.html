Pour requêter les descriptions visuelles supplémentaires de tous les POI : requête POST vers https://super.docker.app/users/sendall.

Pour requêter les descriptions visuelles supplémentaires de POI spécifiques : requête POST vers https://super.docker.app/users/sendlist avec en paramètres :

body: {
  {"postedUser": "<un ou plusieurs ids>" }
}

Pour afficher un ou plusieurs utilisateurs spécifiques, saisissez les identifiants correspondants dans le champ "Rechercher un ou plusieurs utilisateurs".

"Afficher les utilisateurs correspondants" dirigera vers une liste d'utilisateurs depuis laquelle vous pourrez consulter, modifier ou supprimer les différents niveaux de descriptions des utilisateurs.

"Envoyer les utilisateurs correspondants" déclenchera une réponse du serveur organisé comme suit :
